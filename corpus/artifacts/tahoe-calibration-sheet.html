The genuinely unclosable few — no screenshot or doc resolves these:

  1. Squircle exactness — closer than first claimed: the corpus gives Apple's height field y=(1−(1−x)⁴)^¼, and figma-squircle reproduces the vector corner; only a real-time SDF stays approximate.
  2. Dynamic ink for arbitrary APP text — now fully specified (flip @ Y≈0.36 + luminosity vibrancy) and closable for shell chrome + shader-drawn content; only third-party app-drawn glyphs stay out of reach (Apple auto-applies it to system components only).
  3. Exact shadow / focus-ring numbers — Apple ships none; the official Tahoe Figma UI Kit carries the true effect layers if pixel-exact shadows matter.
  4. Dark-mode material colors read through translucent glass, so they shift with the wallpaper behind — treat the dark hexes as representative, not absolute.
